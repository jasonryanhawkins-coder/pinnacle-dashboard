## Setup

### Step 1 — Deploy to GitHub Pages
1. Create a new GitHub repository named `pinnacle-dashboard`
2. Upload `index.html` to the repository
3. Go to Settings → Pages → Source: Deploy from branch → Branch: main → Save
4. Your dashboard will be live at `https://YOUR-USERNAME.github.io/pinnacle-dashboard`

### Step 2 — Get SAM.gov API Key
1. Log into [sam.gov](https://sam.gov)
2. Click your profile → Profile Details
3. Scroll to System Account → Request System Account
4. Select "Public" access and enable "Opportunities Read"
5. Copy your API key

### Step 3 — Use the Dashboard
1. Open your GitHub Pages URL
2. Paste your API key
3. Click Connect & Search
4. Your key is saved in your browser — no need to re-enter

## Usage
- **Search** — filter by NAICS, set-aside, state, keywords
- **Analyze with Claude** — sends contract to Claude AI for fit analysis
- **Draft Response** — Claude drafts your response email automatically
- **View on SAM.gov** — opens the full contract listing
- **Contact Info** — shows contracting officer details
