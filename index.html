How to Get Your SAM.gov API Key

  1. Go to sam.gov and log into your account (create one tomorrow if you haven't yet)
  2. Click your profile icon in the top right corner
  3. Click "Profile Details" from the dropdown
  4. Scroll down to "System Account" section
  5. Click "Request System Account" or "Add System Account"
  6. Fill in the system account name (e.g. "Pinnacle Dashboard") and select "Public" access
  7. Under permissions, select "Opportunities Read" and "Entity Read"
  8. Submit the request — your API key will be displayed immediately or emailed within minutes
  9. Copy the API key and paste it into this dashboard

The API key is completely free. SAM.gov allows 1,000 requests per day on the free tier — more than enough for daily searches.
